# 📌 Overview

The **Java Quiz Application** is a console-based project developed using Java. It allows users to participate in a timed quiz, answer multiple-choice questions, and receive a detailed performance report after completion.

The application also provides a complete answer review showing the user's selected answer, the correct answer, and the status of each question.

---

# ✨ Features

✅ Interactive Quiz Menu

✅ Multiple Choice Questions (MCQ)

✅ 15-Second Countdown Timer

✅ Automatic Timer Display

✅ Input Validation

✅ Instant Question Navigation

✅ Final Score Report

✅ Correct & Wrong Answer Count

✅ Detailed Answer Review

✅ Console-Based User Interface

# 🚀 Application Flow

```text
Start Application
        │
        ▼
Main Menu
        │
        ▼
Quiz Starts
        │
        ▼
Timer Starts (15 Seconds)
        │
        ▼
Answer the Question
        │
        ▼
Next Question
        │
        ▼
Repeat Until All Questions Finish
        │
        ▼
Final Score Report
        │
        ▼
Detailed Answer Review
```
